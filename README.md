# eHotel Management System
A full-stack Java application for managing hotel bookings, rooms, and employees.

## Tech Stack
* **Backend:** Java (Servlets/JSP)
* **Database:** PostgreSQL
* **Frontend:** HTML/CSS

## How to Run
1. Run the scripts in the `/sql` folder to set up the database.
2. Import the project into Eclipse.
3. Update the `DBConnection.java` file with your Postgres password.
